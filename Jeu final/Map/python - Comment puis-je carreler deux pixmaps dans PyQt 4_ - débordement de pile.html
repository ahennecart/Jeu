Comment puis-je carreler deux pixmaps dans PyQt 4?

Poser une question
up vote 0 down vote favori

En commençant par 2 fichiers png sur le disque (que je peux charger sur 2 pixmaps) avec les mêmes dimensions, comment puis-je les découper horizontalement avec un petit espace vertical (transparent) entre eux, comme ceci:

aaaaa bbbbb
aaaaa bbbbb
aaaaa bbbbb

et enregistrer le résultat sous un autre pixmap?

Environnement: PyQt 4.8.3 sous Windows

python windows qt qt4 pyqt4

Cela devrait être un bon point de départ (non testé):

painter = QPainter() // the QPainter to the object you want to paint

a = QImage("file1.png")
b = QImage("file2.png")
space = 5 // pixels
tile = QPixmap(QSize(a.width() + b.width(), a.height() + space)
painter2.setBrush(QBrush(QColor(0, 0, 0, 0)))
painter2.fillRect(0, 0, tile.width(), tile.height())
painter2 = QPainter(tile)
painter2.drawImage(QPoint(0, 0), a)
painter2.drawImage(QPoint(a.width(), 0), b)

painter.setBrush(tile.toImage())

// now anything you draw will be filled with the tiled pattern
